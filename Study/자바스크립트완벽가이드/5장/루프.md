# 루프
 - 루프문은 분기 경로에서 일정 부분의 코드를 실행한 후, 다시 해당 분기 경로로 되돌아 간다.
 - while, do-while, for, for-in이라는 4개의 루프문이 있다.
  - Array 클래스에 특화된 루프 메서드도 존재한다.

## while
```
while(표현식)
 문장
```
 - 가장 기본적인 루프문
 - 표현식을 평가 > 몸체를 형성하는 문장 실행 > 표현식 재평가...
  - 표현식이 false가 될 때까지 반복 실행한다.
